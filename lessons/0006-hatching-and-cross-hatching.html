Lesson 0006: Hatching & Cross-Hatching

Build value gradients using parallel lines and intersecting layers.

Phase 2: Shading & Volume

Lesson Goal Create smooth value gradients from light to dark using hatching and cross-hatching techniques.

The Power of Parallel Lines

Hatching is one of the oldest shading techniques. By drawing parallel lines closer together, you create darker values. By spreading them apart, you create lighter values. It's that simple — and that powerful.

Hatching Practice

1

Draw a long rectangle (about 10 cm × 3 cm). This is your value strip.

2

Start at the left with widely spaced diagonal lines (light). Gradually move them closer together as you go right.

3

By the right end, the lines should be nearly touching (dark). You should see a smooth gradient.

Cross-Hatching Practice

1

Draw another value strip. First layer: diagonal lines (top-left to bottom-right).

2

Second layer: diagonal lines the opposite way (top-right to bottom-left). Add these only on the right half for a darker value.

3

Third layer: vertical lines on the darkest quarter. Each layer doubles the darkness.

Tip Keep the lines straight and parallel. Vary the spacing, not the pressure. Consistent, tidy lines look professional; messy lines look messy.

Mini-Project: Shaded Cylinder & Cube

Draw a cylinder (like a can) and a cube. Shade both using hatching and cross-hatching. Use only line work — no smudging or blending. Aim for 5 distinct value levels on each.

Summary

  • Hatching = parallel lines. Closer spacing = darker value
  • Cross-hatching = multiple layers of hatching at different angles
  • Vary spacing, not pressure, for smooth gradients
  • Keep lines straight and parallel for a clean look
