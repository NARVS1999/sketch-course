Lesson 0003: Basic Geometric Shapes

Circles, squares, triangles, and ellipses — the building blocks of everything you'll draw.

Phase 1: Foundation & Line Control

Lesson Goal Draw circles, squares, triangles, and ellipses confidently and consistently — no compass or ruler.

Why Shapes Matter

Every complex object is just a combination of simple shapes. A head is an ellipse (sphere). A house is a square + triangle. A bottle is a rectangle + curves. If you can draw basic shapes well, you can draw anything.

Circles & Ellipses

1

Draw a circle in one continuous motion. Don't go slow — go smooth. Aim for it to close perfectly.

2

Fill a row with circles. Then draw concentric circles (one inside another) like a target.

3

Draw ellipses — stretched circles. Practice them tall (vertical) and wide (horizontal).

Squares & Triangles

1

Draw a square with four straight sides. Check: are the corners 90 degrees? Are opposite sides parallel?

2

Draw a grid of squares. Then rotate them — draw squares at a tilt (diamond shapes).

3

Draw triangles — equilateral, isosceles, and right triangles. Make the corners sharp and the sides straight.

Common Mistake Beginners draw circles too slowly, which makes them wobble. Speed up slightly — a confident line is smoother than a cautious one, even if it misses the mark. You can always try again.

Quick Quiz

1. How should you draw a circle for the best result?
2. What is an ellipse?

Mini-Project: Shape Combo

Draw a simple still life using only basic shapes. For example: an apple (circle + oval), a box (square + rectangle), and a party hat (triangle + rectangle). Don't worry about details — just get the shapes down.

Summary

  • Every drawing is built from circles, squares, triangles, and ellipses
  • Draw circles in one smooth motion — commit to the line
  • Ellipses are stretched circles; practice both orientations
  • Squares need parallel sides and 90-degree corners
